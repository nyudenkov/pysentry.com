Quick Installation

Get started with your preferred method

Recommended: Run instantly without installation 

# Quick scan
uvx pysentry-rs /path/to/project

# With options
uvx pysentry-rs --sources pypa,osv --format json

Install as Python package (Linux/macOS) 

# Install from PyPI
pip install pysentry-rs

# Run scanner
pysentry-rs /path/to/project

Install with Rust package manager 

# Install from crates.io
cargo install pysentry

# Run scanner
pysentry /path/to/project

Download pre-built binaries 

# Linux/macOS
curl -L https://github.com/nyudenkov/pysentry/releases/latest/download/pysentry-linux-x64.tar.gz | tar -xz
./pysentry-linux-x64/pysentry --help
